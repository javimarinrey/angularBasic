Empezando

Instalaciones mínimas

  • NodeJs
    https://nodejs.org/es/
  • TypeScript
    http://www.typescriptlang.org/
  • Angular CLI
    https://cli.angular.io/
    https://github.com/angular/angular-cli

Generar y arrancar proyecto

  
    ng new PROJECT-NAME
    cd PROJECT-NAME
    ng serve -o

Navegue a http://localhost:4200/. La aplicación se volverá a cargar automáticamente si cambia alguno de los archivos fuente

Puede configurar el host HTTP predeterminado y el puerto utilizado por el servidor de desarrollo con dos opciones de línea de comandos:

  
    ng serve --host 0.0.0.0 --port 4201

Actualizar Angular CLI

Si está utilizando Angular CLI 1.0.0-beta.28 o inferior, debe desinstalar el paquete angular-cli:

  
    npm uninstall -g angular-cli
    npm uninstall --save-dev angular-cli

Para actualizar Angular CLI a una nueva versión, debe actualizar tanto el paquete global como el paquete local de su proyecto.

Global package:

  
    npm uninstall -g @angular/cli
    npm cache clean
    # if npm version is > 5 then use `npm cache verify` to avoid errors (or to avoid using --force)
    npm install -g @angular/cli@latest

Local project package:

  
    rm -rf node_modules dist # use rmdir /S/Q node_modules dist in Windows Command Prompt; use rm -r -fo node_modules,dist in Windows PowerShell
    npm install --save-dev @angular/cli@latest
    npm install

Construir proyecto

  
    ng build
